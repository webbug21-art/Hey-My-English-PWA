# Hey, My English 설치와 학습 순서

## GitHub Pages 업데이트

1. Hey-My-English-PWA.zip의 압축을 풉니다.
2. 안의 index.html, manifest.webmanifest, sw.js, icon.svg를 GitHub 저장소의 같은 위치에 업로드합니다.
3. 같은 이름의 기존 파일이 있으면 그대로 덮어씁니다.
4. Commit changes를 누릅니다.
5. 1~3분 뒤 iPhone Safari에서 GitHub Pages 주소를 엽니다.
6. 공유 → 홈 화면에 추가를 선택합니다.

## 추천 암기 순서

1. 중요도 5의 180문장을 먼저 학습합니다.
2. 한국어만 보고 영어를 말한 뒤 영어 보기를 누릅니다.
3. 쉐도잉 모드에서 색칠된 덩어리별로 따라 말합니다.
4. 원래 팟캐스트를 들으며 같은 패턴이 들리는지 확인합니다.
5. 막힌 문장은 어려움을 눌러 다시 복습합니다.
6. 5점이 익숙해지면 중요도 4의 150문장, 마지막으로 유머·리듬용 3의 30문장을 학습합니다.

## 에피소드 선택

에피소드 필터는 이제 스크립트 번호가 아니라 짧은 주제로 표시됩니다.

1. 취미·친구·진로
2. 꿈·영어·불안
3. 루틴·집중·명상
4. 진짜 친구·경계선
5. 폰·쇼핑·가족 유머
6. 노래 비하인드·증명하기

이번 주 추가 에피소드는 `노래 비하인드·증명하기`입니다. `헤드라인`, `에픽하이 노래`, `증명`, `시간이 판단`, `제약이 연료` 같은 검색어로 찾아볼 수 있습니다.

문장은 안정적인 학습을 위해 원문 표현을 가볍게 정리하거나 재사용 가능한 형태로 변형했습니다. 팟캐스트 또는 제작자와 관련 없는 개인 학습용 비공식 앱입니다.
